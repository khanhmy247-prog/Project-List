# Tiêu chuẩn Phát triển Web

Tài liệu này ghi lại các quy tắc và tiêu chuẩn để tạo ra các trang web chất lượng cao trong dự án này.

## 1. Nguyên tắc Thiết kế
- **Thẩm mỹ hiện đại**: Sử dụng màu sắc hài hòa, gradient nhẹ, và bo góc (border-radius).
- **Responsive**: Đảm bảo giao diện hiển thị tốt trên cả máy tính và thiết bị di động.
- **Micro-animations**: Thêm các hiệu ứng chuyển cảnh nhỏ để tăng trải nghiệm người dùng.

## 2. Quy chuẩn Code (HTML/CSS/JS)
- **Semantic HTML**: Sử dụng các thẻ đúng ý nghĩa (header, footer, main, section).
- **Vanilla CSS**: Ưu tiên sử dụng CSS thuần để có sự linh hoạt tối đa.
- **Cấu trúc file**: Giữ code sạch sẽ, có comment giải thích các phần quan trọng.

## 3. Quy trình Triển khai
- Kiểm tra lỗi hiển thị trên trình duyệt.
- Chạy thử nghiệm trên `localhost` trước khi bàn giao.
